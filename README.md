# FashOJ - 编程竞赛平台

![Static Badge](https://img.shields.io/badge/vue-3-blue)
![Static Badge](https://img.shields.io/badge/go-1.24.1-blue)

## 项目简介

FashOJ 是一个高性能的在线编程竞赛平台，提供题目评测、比赛管理和用户排名功能。

## 功能特性

-   用户认证系统
-   题目管理与提交
-   实时评测与结果返回
-   比赛创建与管理
-   用户排名系统

## 技术栈

-   后端: Go (Gin 框架)
-   判题机: Go 独立模块
-   前端: Vue.js
-   数据库: MySQL

## 快速开始

```bash
# 克隆项目
git clone https://github.com/your-repo/FashOJ.git
cd FashOJ

# 启动服务
./scripts/deploy.sh
```

-   C++
